Simulation Report – Basic ALU

Objective:
Design a 4-bit ALU supporting Addition, Subtraction, AND, OR and NOT.

Inputs:
A = 1010
B = 0011

Results:
000 → Addition → 1101
001 → Subtraction → 0111
010 → AND → 0010
011 → OR → 1011
100 → NOT → 0101

Conclusion:
The ALU successfully performs arithmetic and logical operations.
