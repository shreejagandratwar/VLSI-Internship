# CODTECH Internship - Verilog Projects

## Overview
This repository contains Verilog-based digital design projects completed during the internship, including processor design, memory modules, ALU implementation, and digital filter design.

## Tasks Completed-----

### Task 1: ALU Design
- Arithmetic and logical operations
- Simulation verification

### Task 2: RAM Design
- Synchronous RAM with read/write operations
- Functional simulation

### Task 3: Pipeline Processor Design
- 4-stage pipelined processor
- ADD, SUB, LOAD instructions

### Task 4: Digital FIR Filter Design
- FIR filter implementation
- Performance analysis and waveform verification

## Tools Used
- Verilog HDL
- Icarus Verilog
- GTKWave

Successfully designed, simulated, and analyzed digital systems using Verilog.
